Conditional Code Suite allows you to add styles and scripts to specific pages or post types without having to touch your theme\'s files.

== Description ==
Add custom code to your site without touching your theme:

- Custom body classes
- wp_head code
- wp_footer code
- wp_body_open code

This can be controlled by conditions:

- Page
- Post type

== Installation ==
1. Install from the plugins menu or upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
